# 1.0.2 What You'll Learn in the Module?

> [!abstract] Module Overview
> This module focuses on the fundamental forms and structures of data. We will also experiment with the process of transforming raw data into visual elements to facilitate better interpretation and storytelling.

## Module Title
**Experience Analytics**

## Overall Objective
Analyze the characteristics and basic functions of data analysis.

---

## Learning Roadmap

The following table outlines the specific topics covered in this module, alongside their measurable learning objectives.

| Topic | Learning Objective |
|:---|:---|
| What is Data? | Identify the applications of data in everyday life. |
| Data Surrounds Us | Compare and contrast different types of data. |
| Business Understanding | Evaluate the value obtained through data analysis. |

---

> [!tip] Key Takeaway
> By the end of this module, you will be able to distinguish between different data types (structured, unstructured, semi-structured) and understand how raw data translates into actionable business insights.
