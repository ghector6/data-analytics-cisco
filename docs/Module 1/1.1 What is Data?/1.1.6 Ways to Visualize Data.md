# 1.1.6 Ways to Visualize Data

> [!abstract] Introduction
> Data visualization is the graphical representation of information. Different chart types serve different purposes, and choosing the right one is essential for effectively communicating insights. This note covers the most common visualization types: line charts, column charts, bar charts, pie charts, and scatter plots.

---

## 1. Line Charts

Line charts are among the most commonly used visualization types. They are particularly effective for comparing two datasets and showing changes over time, especially when the number of data points is high.

### What is a Line Chart?

A line chart displays information as a series of data points connected by straight line segments. It reveals trends, patterns, and relationships in data over a continuous interval.

### Key Characteristics

- **X-Axis**: Typically displays time (days, weeks, months, years)
- **Y-Axis**: Displays the measured values or variables
- **Data Points**: Individual values plotted on the chart
- **Line Segments**: Connect data points to show progression and trends

### When to Use Line Charts

Line charts excel when:

- **Time is a primary variable**: Showing trends, progressions, or historical patterns
- **Data points are numerous**: Handling large datasets with many observations
- **Comparing multiple datasets**: Overlaying two or more series for comparison
- **Identifying patterns**: Revealing seasonal trends, growth, or decline

### Common Use Cases

| Industry | Use Case |
|:---|:---|
| Business | Quarterly sales performance over 5 years |
| Business | Customer growth per week during a store's first year |
| Finance | Stock price changes from opening to closing bell |
| Science | Temperature changes over time |
| Marketing | Website traffic monitoring |

### Best Practices

| Practice | Why It Matters |
|:---|:---|
| Label axes and reference lines | Provides context and makes data interpretable |
| Use continuous lines | Helps readers follow trends and progression |
| Limit to 5 lines maximum | Prevents visual clutter and confusion |
| Include a clear legend | Ensures each dataset is identifiable |
| Always add a title | Communicates the chart's purpose at a glance |

---

## 2. Column Charts

Column charts are among the most common visualization types. They represent data using vertical bars, where the height of each bar corresponds to a numeric value for a specific data point. They are ideal for comparing values across similar categories.

### What is a Column Chart?

A column chart displays data as rectangular bars with heights proportional to the values they represent. The vertical orientation makes it easy to compare categories side by side.

### When to Use Column Charts

Column charts are ideal when:

- **Comparing categories**: Showing values across different groups
- **Tracking discrete data points**: Displaying individual measurements
- **Emphasizing magnitude**: Highlighting differences between values

### Common Use Cases

| Use Case | Description |
|:---|:---|
| Revenue by country | Comparing income across different nations |
| Annual sales | Tracking yearly performance of specific companies |
| Student scores | Comparing average test scores across classes |

### Best Practices

| Practice | Why It Matters |
|:---|:---|
| Label axes clearly | Ensures data is interpretable |
| Show time on X-axis | When tracking changes over time, place time on the horizontal axis |
| Sort by height | Shows trends and rankings at a glance |
| Maximum 7 columns | Prevents visual clutter; use multiple charts if more are needed |
| Start Y-axis at 0 | Prevents misleading visual comparisons |
| Space between columns | Approximately half the width of a column for clear separation |

---

## 3. Bar Charts

Bar charts are similar to column charts, except the data is displayed horizontally. This orientation allows for simple comparison across multiple data points, especially when category labels are long.

### What is a Bar Chart?

A bar chart displays data as horizontal bars, with the length of each bar proportional to the value it represents. The horizontal orientation makes it ideal for comparing many categories or displaying long category names.

### When to Use Bar Charts

Bar charts are ideal when:

- **Comparing many categories**: Displaying numerous data points
- **Category labels are long**: Horizontal orientation accommodates longer text
- **Ranking data**: Showing order or hierarchy clearly

### Common Use Cases

| Use Case | Description |
|:---|:---|
| GDP comparison | Comparing gross domestic product across countries |
| Sales by salesperson | Tracking individual sales performance |
| Student grades | Comparing grades across multiple students |

### Best Practices

| Practice | Why It Matters |
|:---|:---|
| Label axes clearly | Ensures data is interpretable |
| Sort bars by value | Shows rankings and trends at a glance |
| Start X-axis at 0 | Prevents misleading visual comparisons |
| Use consistent colors | Helps readers follow the data easily |

---

## 4. Pie Charts (Pie or Circular Charts)

Pie charts show parts of a whole. Each slice or segment represents a percentage of the total, and the sum of all segments must equal 100%.

### What is a Pie Chart?

A pie chart is a circular chart divided into slices to illustrate numerical proportion. Each slice represents a category's contribution to the whole.

### When to Use Pie Charts

Pie charts are ideal when:

- **Showing proportions**: Displaying how parts make up a whole
- **Limited categories**: Best with a small number of segments
- **Emphasizing percentages**: Highlighting relative contributions

### Common Use Cases

| Use Case | Description |
|:---|:---|
| Annual expense categories | Visualizing budget allocation |
| Energy sources | Showing energy mix by source |
| Survey results | Displaying response distribution |

### Best Practices

| Practice | Why It Matters |
|:---|:---|
| Keep categories to a minimum | Too many slices become unreadable |
| Use different colors | Helps distinguish each segment |
| Sort by size | Largest segments first for clarity |
| Ensure total is 100% | Validates the proportion representation |

---

## 5. Scatter Plots

Scatter plots are used to visualize correlations or to demonstrate distribution. They are excellent for showing groupings or identifying outliers.

### What is a Scatter Plot?

A scatter plot uses Cartesian coordinates to display values for two variables. Each point represents an observation, and the position on the X and Y axes shows the relationship between the two variables.

### When to Use Scatter Plots

Scatter plots are ideal when:

- **Showing correlation**: Identifying relationships between two variables
- **Demonstrating distribution**: Visualizing data spread and density
- **Identifying outliers**: Spotting anomalies in the data
- **Revealing groupings**: Showing natural clusters in the data

### Common Use Cases

| Use Case | Description |
|:---|:---|
| Price vs. quality | Comparing product value |
| Height vs. weight | Analyzing physical correlations |
| Temperature vs. sales | Finding seasonal patterns |
| Marketing spend vs. revenue | Measuring campaign effectiveness |

### Best Practices

| Practice | Why It Matters |
|:---|:---|
| Label axes clearly | Ensures data is interpretable |
| Use a sufficiently large dataset | More points reveal more meaningful patterns |
| Limit to 2 trend lines | Too many trends create confusion |
| Use distinct colors for groups | Makes groupings easier to identify |

---

## Visualization Type Comparison Summary

| Chart Type | Best For | Key Limitation |
|:---|:---|:---|
| **Line Chart** | Trends over time, continuous data | Limited to 5 lines for readability |
| **Column Chart** | Comparing categories vertically | Maximum 7 columns for readability |
| **Bar Chart** | Comparing categories horizontally | Limited space for many bars |
| **Pie Chart** | Showing proportions | Best with limited categories (5 or fewer) |
| **Scatter Plot** | Correlation and distribution | Requires large datasets for meaningful insights |

---

> [!tip] Key Takeaway
> Choosing the right visualization depends on the story you want to tell. Line charts reveal trends over time, column and bar charts compare categories, pie charts show proportions, and scatter plots expose correlations. Master each type to communicate insights effectively.
